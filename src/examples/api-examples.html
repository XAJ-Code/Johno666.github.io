运行时API示例 ​

本页演示了VitePress提供的一些运行时API的用法。

该useData（） API可用于访问当前页面的网站、主题和页面数据。它在“.md”和“.vue”文件中都有效：

md
<script setup>
import { useData } from 'vitepress'

const { theme, page, frontmatter } = useData()
</script>

## Results

### Theme Data
<pre>{{ theme }}</pre>

### Page Data
<pre>{{ page }}</pre>

### Page Frontmatter
<pre>{{ frontmatter }}</pre>
